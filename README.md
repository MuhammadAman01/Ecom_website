# 🛍️ E-Commerce Platform

A modern and scalable e-commerce web application built with **Next.js** and **Supabase**. The platform provides a seamless shopping experience for customers along with a powerful admin dashboard for managing products, orders, users, and inventory.

## ✨ Features

### 👤 User Panel
- User authentication (Sign Up / Login)
-all product at one page
- Product details page
- Shopping cart
- Secure checkout process
- Order history
- Responsive design for all devices

### 🛠️ Admin Panel
- Secure admin authentication
- Dashboard overview
- Add, edit, and delete products
- Manage categories
- View and manage customer orders
- Manage users
- Inventory management

## 🚀 Tech Stack

- **Frontend:** Next.js
- **Backend:** Supabase
- **Database:** Supabase PostgreSQL
- **Authentication:** Supabase Auth
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

## 📦 Installation

```bash
git clone https://github.com/MuhammadAman01/Ecom_website.git
cd your-repository
npm install
npm run dev
```

## 🔑 Environment Variables

Create a `.env.local` file and add:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```
